# CV ATS v1

Primera version Markdown orientada a:

- Maintenance Planner / Planificador de Mantenimiento.
- Oficina Tecnica de Mantenimiento.
- Material Controller / Repuestos / Abastecimiento tecnico.

## Archivos

- `CV_ATS_Planner_Oficina_Tecnica_ES.md`: CV ATS v1 en espanol, una columna, sin telefono/email, sin edad, sin salario, sin foto, sin tablas y sin iconos.
- `CV_ATS_Profile_Summary_EN.md`: resumen profesional en ingles para futura version internacional.

## Reglas aplicadas

- No se genero PDF, DOCX ni HTML.
- No se tocaron archivos en `outputs/`.
- No se publico telefono/email.
- No se publicaron clientes de Garage365.
- No se incluyo pretension salarial.
- No se declaro un modulo SAP especifico.
- No se presento mineria como experiencia directa de perforacion.
- No se presento metodologia formal de confiabilidad.
